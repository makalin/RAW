# Changelog

All notable changes to RAW will be documented in this file.

## [1.0.0] - 2024-01-01

### Added
- Core framework with signals and reactive DOM
- `state()` - Create reactive signals
- `html` - Tagged template literal for components
- `mount()` - Mount components to DOM
- `computed()` - Derived reactive values
- `effect()` - Side effects that auto-run
- Form handling utilities with validation
- Router for SPA navigation
- Global state management (store pattern)
- Developer tools and debugging utilities
- Utility functions (debounce, throttle, memo, etc.)
- Comprehensive TypeScript support
- Testing framework setup (Vitest)
- CI/CD pipeline (GitHub Actions)
- Documentation (API reference and guides)
- Example projects (counter, todo, router, store)

### Features
- Zero Virtual DOM
- Zero Hydration
- Surgical DOM updates via signals
- Type-safe API
- Small bundle size (~1kb core)
- Fast performance

# Assets

This directory contains project assets including logos and images.

## Logo

- `logo.svg` - RAW framework logo (SVG format)
- `logo.png` - RAW framework logo (PNG format) - Add your PNG logo here

## Usage

The logo is referenced in the main README.md file. To add a PNG version:

1. Place your `logo.png` file in this directory
2. Update README.md to reference `./assets/logo.png` instead of or alongside the SVG
